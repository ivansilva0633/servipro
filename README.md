# ServiPro - Gestao para prestadores de servico do bairro

## Rodar local
pip install -r requirements.txt
python app.py  ->  http://localhost:5000

## Subir no ar GRATIS (Render.com)
1. Conta gratis em render.com
2. Suba estes arquivos no GitHub
3. New > Web Service > conecte o repo
4. Start command: gunicorn app:app
5. Deploy -> link gratis tipo servipro.onrender.com
